The code option

Define a code template tpl:

tpl    = 'lm(mpg ~ %s, data = mtcars) |> summary() |> coef()'
x_vars = names(mtcars)[2:4]

We run regressions on three variables one by one:

lm(mpg ~ cyl, data = mtcars) |> summary() |> coef()
